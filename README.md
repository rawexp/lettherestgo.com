# lettherestgo.com

Landing page for [Let The Rest Go](https://lettherestgo.com) — a brutally simple iOS app for focus-driven task management.

## About

Single-page static site. No build step, no dependencies.

## Deploy

Push to `main` and enable GitHub Pages (or drop on any static host).

index.html    # The whole thing
